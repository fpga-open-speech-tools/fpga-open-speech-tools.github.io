Microphone Array

Overview

The Microphone Array is a 16 Element MEMS Microphone Array. The microphones are place in a four by four grid with 25mm spacing. The Microphone Array send data to the Audio Blade or Audio Research by converting a the microphone data into a serial data stream, encoded by the Max V CPLD. The Audio Blade and the Microphone Array communicate using differential signaling over an RJ45 Cable. 3.3V to 12V Power is supplied to the Microphone Array using the RJ45 Cable, as well.
